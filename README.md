
Minefading
===========================================
Full title: Relics of the Fading City Special: Minefading
Chinese title: 《亡都遗骨》特别篇：我的余晖

Project overview:
This repository is the Forge mod workspace for Minefading.
The current baseline keeps only the core mod bootstrap and configuration wiring,
so feature content can be added on top of a clean project skeleton.

Environment:
- Minecraft 1.20.1
- Forge 47.4.10
- Java 17

Common commands:
- gradlew genIntellijRuns
- gradlew genEclipseRuns
- gradlew runClient
- gradlew build
